Create new Application

You are about to create a new application to be hosted in the UNINETT WebApp Park.

First, you need to register an app identifier. This will be used as part of the domain name. It can only contain letters and numbers [a-z] and [0-9].

Your application will be accessible from:
https://<identifier>.uwap.org

Then, give your application a short desriptive name, and a longer descriptive text to explain what your application does.
